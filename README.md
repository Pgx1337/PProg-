
# 🎰 KASYNO ONLINE 

Nowoczesna platforma kasyna internetowego stworzona z myślą o wydajności, bezpieczeństwie i świetnym doświadczeniu użytkownika.
Nasz projekt jest oparty na instutuacjach oferujących gry hazardowe, totalizatorach sportowych. Nasza strona zawiera gry takie jak ruletka, blackjack, poker, baccarad, slotsy etc.
Użytkownik może wpłacić pieniądze następnie grać za nie, wygrać lub przegrać, dorobić się lub stracić


## 🚀 Funkcje

* 🎲 Gry kasynowe (sloty, blackjack, ruletka)
* 👤 System użytkowników (rejestracja, logowanie)
* 💰 System salda i transakcji
* 🔒 Bezpieczna autoryzacja i przechowywanie danych
* 📊 Panel administracyjny
* 🌐 Responsywny design (desktop)


## 🛠️ Technologie

Projekt wykorzystuje:

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js 
* Baza danych: MySQL
