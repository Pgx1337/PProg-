# Projekt o nazwie kasyno.
Nasz projekt jest oparty na instutuacjach oferujących gry hazardowe, totalizatorach sportowych. Nasza strona zawiera gry takie jak ruletka, blackjack, poker, baccarad, slotsy etc.
Użytkownik może wpłacić pieniądze następnie grać za nie, wygrać lub przegrać, dorobić się lub stracić


Plan na 2tyg.:
-szkic,
-zrobienie html i css,
-ekran startowy,
-Historia gier i transakcji,
-balans, wplaty wyplaty, 
-obrazki, wyglad, uklad (css),
-logowanie, database, autoryzacja,
